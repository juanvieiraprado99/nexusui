# Table API

## `table[n-table]`

| Input | Type | Default | Description |
|---|---|---|---|
| `nVariant` | `'default' \| 'striped' \| 'bordered'` | `'default'` | Visual style variant |
| `nSize` | `'default' \| 'compact' \| 'comfortable'` | `'default'` | Cell padding size |
| `nClass` | `string` | `''` | Extra Tailwind classes |

## `n-table-scroll`

Responsive wrapper with `overflow-x-auto`. Wrap any `table[n-table]` to enable horizontal scrolling on narrow viewports.

| Input | Type | Default | Description |
|---|---|---|---|
| `nClass` | `string` | `''` | Extra Tailwind classes |
